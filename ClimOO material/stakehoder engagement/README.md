# ClimOO Ontology suite -  Stakehoder Engagement


## Description

This directory contains resources used in engaging with stakeholders from the [ClimOO project](https://pthsta.it/climoo), along with summaries of meeting conducted with stakeholders for documentation purposes. All materials have been anonymized and edited to remove any personal or confidential information. Stakeholders revised and consented to make these materials publicly available. 

Materials include:

- A list of the questions used to drive the initial discussions with stakeholders, in docx and txt formats (as of 2026-04-22).
- Summaries of the meeting conducted with stakeholders, in docx and PDF formats (as of 2026-04-22).
  - Meetings have been numbered in ascending order to reflect the chronological sequence in which discussions have taken place.
  - Summaries have been collected in the subdirectory named "meeting summaries".

## Legenda

For reasons of confidentiality, institutes involved in the ClimOO Project are refered as Institutes A, B, and C, and related representatives as A1, A2, etc.
