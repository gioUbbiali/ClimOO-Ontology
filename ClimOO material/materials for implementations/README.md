# Materials for implementation


This folder contains materials for implementing ClimOO, including:

- A list of ontologies from which ClimOO currently reuses or adapts classes and object properties and relevant ontologies from which we may import in the future (ClimOO imports), in Excel format.
- A list of the terms that ClimOO currently covers or that we aim to include with further implementations, in Exce (currently) and PDF (coming oon) formats.
