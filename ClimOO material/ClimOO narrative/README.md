# ClimOO Narrative


This folder contains the shared narrative on which the ClimOO Ontology suite has been designed. The narrative has been defined by engaging with [ClimOO project](https://pthsta.it/climoo) stakeholders, according to the guidelines of Ubbiali (2025). The narrative is rendered in PDF and DOCX formats.

### *DISCLAIMER: The narrative is a work in progress an subjected to changes* 

## Bibliography

Ubbiali, G. A. (2025). Designing Foundational Strategies for Developing a Family of Food Systems Sustainability Ontologies. PhD Thesis, UNIMI. https://air.unimi.it/handle/2434/1161515
