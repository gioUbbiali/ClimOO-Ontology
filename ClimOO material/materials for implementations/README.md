# Materials for implementation


This folder contains materials for implementing ClimOO, including:

- A list of ontologies from which the ClimOO Ontology suite currently reuses or adapts classes and object properties and relevant ontologies from which we may import in the future (ClimOO imports), in Excel format.
- A list of the terms that the ClimOO Ontology suite currently covers or that we aim to include with further implementations, in Excell (currently) and PDF (coming oon) formats.
- The protocol followed to develop [ClimOO-SDC2](https://w3id.org/climoo/climoo-sdc2) from [SDC Onto](https://github.com/hbabaie1/Sustainable-Development-and-Climate-SDC-ontology/blob/main/SDC_v1.owl), in PDF and Docx formats.
