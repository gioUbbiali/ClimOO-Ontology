# ClimOO Ontology - Initial Core Ontological Module

STATUS: Deprecated.
JUSTIFICATION: It contains imports from existing ontologies that have been insufficiently organized and a preliminary draft of some classes of the ClimOO Ontology. Despite that, it was useful to give an overview of the ontology focus to stakeholders at an initial stage of the work.

## Description

This folder contains the initial core ontological module of the ClimOO Ontology. This module, complemented by the [SDC-Ontology import file](https://github.com/ClimOO/ClimOO-Ontology/blob/main/ClimOO%20material/src/ontology/imports/sdc-ontology-imports.ttl), constitutes the ontology starting point that we presented to project stakeholders and on which we built up the ClimOO Ontology. The module is rendered in [Terse RDF Triple Language (TTL)](https://www.w3.org/TR/turtle/).
